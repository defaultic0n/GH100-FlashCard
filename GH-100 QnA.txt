1. Which two pieces of information are included in your organization's log?
Select only one answer.
A.  The user that performed the action and the date and time of the action.
B.  The date and time of the action and changes in permissions.
C.  Changes in permissions and users being promoted to administrators.
D.  Users being promoted to administrators and the user that performed the action.

Answer: A
Explanation: Logs typically include the user who performed the action and the timestamp


2. What file should you use to create documentation for collaborators that lists supported versions of the project? Select only one answer.
A.  SECURITY.md
B.  CONTRIBUTING.md
C.  SUPPORT.md
D.  .gitignore file

Answer: A
Explanation: SECURITY.md is used to document security-related information, including supported versions.


3. What tool should you use to automate part of your security process? Select only one answer.
A.  Add Dependabot to your code base.
B.  Add access restrictions to your enterprise.
C.  Create security documentation.
D.  Add advisories for your users.

Answer: A
Explanation: Dependabot automates dependency updates to address security vulnerabilities


4. Which two pieces of information should be included in a security advisory? Select only one answer.
A.  Product affected and severity.
B.  Severity and exposure list.
C.  Administrator name and severity.
D.  Exposures list and administrator name.

Answer: A
Explanation: A security advisory should include the affected product and the severity of the issue
